Principais formatações

Negrito / Destaque

usando a tag b, faço o texto ficar em Negrito mas não é semântico.
Exemplo: Negrito

Usando a tag STRONG, faço o texto ficar em Negrito mas com semântica.( Texto em Destaque)
Exemplo: Texto em Destaque

Itálico / Ênfase

Usando a tag i, o texto fica em Itálico mas não é semântico
Exemplo: Itálico

Usando a tag em, o texto fica em Ênfase (e itálico)(Forma Semântica)
Exemplo: Texto em Ênfase

Texto Marcado

Podemos criar tambem um texto marcado usando a tag MARK

Texto Grande e Pequeno

Tag big faz o texto ficar Grande (Não Semântica)

Tag small faz o texto ficar Pequeno

Texto Excluído

Com a tag del, o Texto é marcado como excluído; deve ser lido mas desconsiderado

Texto Inserido

Com a tag ins, o texto é marcado como inserido(fica sublinhado)
serve para dar Ênfase que o texto foi inserido posteriormente

A tag U tambem deixa o texto sublinhado mas de forma (Não Semântica)

Texto Sobrescrito

Tag sup faz com que parte do texto fique Sobrescrito. Ex: x23 + y9

Texto Subscrito

Tag sub faz com que parte do texto fique Subscrito. Ex: H2O
